# IronixPay Examples

Official integration examples for [IronixPay](https://ironixpay.com) — accept USDT payments on 7 blockchains.

## Examples

| Example | Stack | Description |
|:---|:---|:---|
| [nextjs-starter](./nextjs-starter) | Next.js 15 + TypeScript | Full checkout flow with webhook verification |

## Quick Start

Pick an example, follow its README, and you'll be accepting payments in 5 minutes.

```bash
git clone https://github.com/IronixPay/ironixpay-examples.git
cd ironixpay-examples/nextjs-starter
cp .env.example .env.local
# Add your API key → npm install → npm run dev
```
